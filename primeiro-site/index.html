Marca texto

Marcações de texto com o exemplo a seguir: animi reprehenderit fugit voluptatibus.

Texto Desconsiderado

Quando o texto deve ser lido, porém descondiderado: asperiores dolores quo in commodi, corrupti aspernatur velit!

Texto Inserido

Quando o pedaço do texto deve ser lido com atenção: Commodi minus eos doloribus, quibusdam voluptatum suscipit

Texto pequeno

Usado para pequenas linhas de contratos: Foasj lkizokoas zeknis nit soltium. Facere quod possimus
